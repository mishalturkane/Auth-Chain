How AuthChain works

  1. Upload: Employer or university uploads certificate PDF/image.
  2. Extract: OCR extracts fields; AI analyzes visuals for tampering.
  3. Verify: Hash/QR checks + registry cross-match (or blockchain anchor check).
  4. Report: System returns Valid / Suspicious / Forged with evidence package.
